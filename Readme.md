# furkot-tomtom-ov2

Generate TomTom OV2 files from [Furkot] trip data.

## Install

```sh
$ npm install --save furkot-tomtom-ov2
```

## Usage

```js
var ov2 = require('furkot-tomtom-ov2');

ov2(trip);
```
